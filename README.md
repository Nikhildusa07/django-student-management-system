# Django Student Management System

A beginner-friendly Django project built as part of my Full Stack Python learning journey.

## Project Overview

This project demonstrates the fundamentals of Django by building a Student Management System with complete CRUD (Create, Read, Update, Delete) functionality.

The main objective of this project is to understand how Django works internally by implementing routing, views, templates, models, ORM, and database operations.

## Features

- Student Registration
- View All Students
- Update Student Details
- Delete Student Records
- Django Admin Panel
- SQLite Database
- Django ORM Integration

## Concepts Practiced

- Django Project & App Structure
- URL Routing
- Function-Based Views
- Templates
- Template Variables
- HTML Forms
- GET & POST Requests
- CSRF Protection
- Django Models
- Migrations
- SQLite Database
- Django ORM
- CRUD Operations
- Django Admin

## Technologies Used

- Python
- Django
- SQLite
- HTML

## Learning Purpose

This project was developed for learning Django fundamentals and understanding how CRUD applications are built using Django's ORM and MVC architecture. It serves as a practice project and a foundation for building larger Django applications.
